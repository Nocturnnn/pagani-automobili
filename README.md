💻 Quer rodar o projeto localmente?

1️⃣ Clone o repositório

git clone SEU_REPOSITORIO_AQUI

2️⃣ Entre na pasta

cd nome-do-projeto

3️⃣ Instale as dependências

npm install

4️⃣ Rode o projeto

npm run dev

5️⃣ Abra no navegador

http://localhost:3000

⚠️ Observação:
O modelo 3D utilizado está em GLB, então caso você queira trocar o carro basta substituir o arquivo dentro da pasta:
/public/models/

eu mesmo usei esse site para modelos 3d: https://sketchfab.com/
obs: você deve renomear o arquivo igual está o modelo original
