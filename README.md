🚀 Rodando o projeto localmente

Siga os passos abaixo para executar o projeto em sua máquina.

1️⃣ Clone o repositório

Abra o terminal e execute o comando abaixo:

git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git

Esse comando irá baixar todo o código do projeto para sua máquina.

Após executar, uma nova pasta com o nome do repositório será criada no diretório atual.

2️⃣ Acesse a pasta do projeto

Entre na pasta que foi criada pelo comando anterior:

cd NOME-DO-REPOSITORIO
3️⃣ Instale as dependências

Este projeto utiliza Node.js e npm para gerenciar pacotes.

Execute:

npm install

Esse comando irá instalar todas as bibliotecas necessárias para o funcionamento do projeto, incluindo:

Next.js

React

Three.js

React Three Fiber

GSAP

Drei

4️⃣ Inicie o servidor de desenvolvimento

Agora rode:

npm run dev

O projeto será iniciado em modo de desenvolvimento.

5️⃣ Abra o projeto no navegador

Acesse o endereço:

http://localhost:3000

Você verá a aplicação rodando localmente.

📦 Modelo 3D

O projeto utiliza um modelo 3D no formato .glb localizado em:

/public/models/

Caso queira substituir o carro por outro modelo:

Baixe um modelo .glb

Substitua o arquivo dentro dessa pasta

Renomeie o arquivo exatamente com o mesmo nome do modelo original

Isso evita precisar alterar o código.

🎨 Onde encontrar modelos 3D

Modelos semelhantes podem ser encontrados em:

👉 https://sketchfab.com/

Node.js 18+
